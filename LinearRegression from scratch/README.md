# Linear Regression From Scratch

## Overview

This project demonstrates the implementation of **Simple Linear Regression from scratch using Gradient Descent** in Python without using machine learning libraries such as Scikit-learn for model training.

The goal of this project is to understand the mathematical concepts behind linear regression and how gradient descent optimizes model parameters.

---

## Features

* Linear Regression implemented from scratch
* Gradient Descent optimization
* Cost function calculation
* Prediction on new data
* Mean Squared Error (MSE)
* Mean Absolute Error (MAE)
* R² Score evaluation
* Cost convergence visualization
* Regression line plotting using Matplotlib

## Mathematical Model

The linear regression equation is:

```
y = θ₀ + θ₁x
```

The model parameters are optimized using Gradient Descent to minimize the Mean Squared Error cost function.
