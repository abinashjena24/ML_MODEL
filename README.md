# Machine Learning Models

A collection of Machine Learning algorithms implemented from scratch and using Python libraries for learning and educational purposes.

## Repository Structure

| Model                           | Description                                                |
| ------------------------------- | ---------------------------------------------------------- |
| 📈 Linear Regression            | Implementation of Linear Regression using Gradient Descent |
| 📊 Logistic Regression          | Binary classification model                                |
| 🌳 Decision Tree                | Tree-based classification and regression                   |
| 🌲 Random Forest                | Ensemble learning using multiple decision trees            |
| 👥 K-Nearest Neighbors (KNN)    | Instance-based learning algorithm                          |
| ✈️ Support Vector Machine (SVM) | Maximum margin classifier                                  |
| 🎯 Naive Bayes                  | Probabilistic classification algorithm                     |
| 🔵 K-Means Clustering           | Unsupervised clustering algorithm                          |
